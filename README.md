## FSL colormap

The function creates an FSLview RGB colormap with n colors. For example

```
map = fsl_colormap(10)

map =

         0    1.0000    1.0000
         0    0.7778    1.0000
         0    0.5556    1.0000
         0    0.3333    1.0000
         0    0.1111    1.0000
    1.0000    0.1111         0
    1.0000    0.3333         0
    1.0000    0.5556         0
    1.0000    0.7778         0
    1.0000    1.0000         0
```
